# DialogSystem.Unity
다이얼로그 시스템에 대해 배운 내용을 저장해놓았습니다.  
#### 1021.
대화창을 UI로 구성하는 방식과 해당 UI를 이용해 대화 시스템을 구축하고 또 그 대화 시스템을 이용해 퀘스트를 구현했습니다.  
자료구조는 ___Dictionary___ 를 주로 사용하였으며 퀘스트, 오브젝트들의 ID에 따라 다른 대화를 하도록 유도했습니다.
